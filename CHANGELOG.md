# Changelog

## v1.0

Initial public release of SevEngine.

### Added

- Mode 7 floor renderer
- Infinite tiled texture rendering
- Skybox support
- Adjustable FOV
- Distance shading
- WASD / Arrow key movement
- FPS counter
- Configurable engine settings
- Asset folder structure

### Issues

- Optimization is poor and leads to low FPS rates. Performance is almost completely dependent on PC specs.
- Resolution is very low.

**These issues are expected to be solved in the next update.**

### Expected in Future Updates

- Player sprites
- Collision management
- Billboard sprite support
- Fullscreen options and higher resolutions
- Further performance optimization
